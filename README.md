# Download Manager Pro

一个轻量、高性能的纯原生下载管理面板，支持多节点 Aria2 集群管理与 OpenList 云盘深度集成。

本项目采用 Node.js 作为后端核心，前端拒绝臃肿的框架，完全基于原生 HTML/CSS/JS 构建。不仅拥有极低的资源占用，更具备堪比原生 App 的丝滑交互体验与极客级的可靠性。

## ✨ 核心特性

### 🚀 极致性能与交互美学
- **无感局部热更新**：前端采用定制的 DOM Diff 算法与 WebSocket 实时通信，拒绝全局重绘，页面状态每秒刷新亦无任何闪烁。
- **原生级视觉动效**：文件列表采用定制的阻尼缓动曲线（Cubic-Bezier）与交错式瀑布流（Staggered）进场动画，纵享德芙般丝滑。
- **全系自然数字排序**：打破反人类的字典序，完全支持自然语言逻辑排序（如 `1, 2, 10` 而非 `1, 10, 2`）。

### 🟣 多节点 Aria2 集群管理
- 支持绑定并管理多个独立的 Aria2 RPC 节点，实时监控各节点上下行网速与运行状态。
- 提供可视化与 JSON 源码双模式的引擎底层参数调优界面，支持热重载。
- **绝对免疫冲突**：底层采用独立组合键（UID）机制，彻底解决多节点环境下的 GID 冲突与任务“诈尸”隐患。

### ☁️ 云盘深度集成 (OpenList)
- 跨端无缝融合：直接在面板内穿透浏览云端目录。
- **高级文件操作**：
  - **批量推送**：支持将云端文件队列式推送至指定的 Aria2 节点进行下载。
  - **安全感知**：内置实时多选计数器，批量操作告别盲目。
  - **高阶重命名**：支持“查找替换”与“正则序列化”模式，并首创**实时变更预览视窗**，所见即所得。

### 🛡️ 工业级稳定架构
- **防爆流处理**：优化 Node.js 后端负载上限，轻松吃下包含成百上千个文件的巨型 BT 种子解析。
- **底层降维打击**：直接通过 HTTP 请求头强算 `Content-Length`，拦截并处理 Aria2 底层时序 Bug。
- 内置极客开发者工具（Debug Panel），实时监控全链路 API 瀑布流与心跳包状态。

## 📦 一键安装与部署

**系统要求**: Debian/Ubuntu 或 CentOS/RedHat 等主流 Linux 发行版 (需 root 权限)

无需繁琐的环境配置，直接在服务器终端运行以下一键安装脚本：

```bash
bash <(curl -sL [https://raw.githubusercontent.com/MechanicF/Download-Manager/main/manager.sh](https://raw.githubusercontent.com/MechanicF/Download-Manager/main/manager.sh))
```
*(注：如果你的默认分支为 `master`，请将链接中的 `main` 替换为 `master`)*

安装过程中，脚本将自动配置 Node.js 20+ 环境、核心依赖，并接管进程守护（PM2）与开机自启机制。

---

## 🕹️ 终端控制台 (`dm` 命令)

安装完成后，你无需记住任何安装路径。在服务器的**任意目录**下，直接输入 `dm` 即可呼出专属的可视化管理菜单：

```bash
dm
```

**菜单功能包含：**
- ▶️ 启动 / ⏹️ 停止 / 🔄 重启 面板服务
- 📝 查看实时运行日志 (排障利器)
- ⚙️ 修复/配置开机自动启动
- ⬇️ **一键从 GitHub 更新至最新版本**
- 🗑️ 彻底安全卸载

## ⚙️ 初始登录与安全建议

- 部署成功后，默认访问地址为：`http://服务器IP:1111`
- **默认账号**: `admin`
- **默认密码**: `password`
- ⚠️ **安全警告**：登录后，请立即前往侧边栏的 **“全局设置”** 中修改默认登录密码，并配置你的 Aria2 RPC 节点和云盘信息！

## 🛠️ 技术栈简述

- **前端**: HTML5 / CSS3 / Vanilla JavaScript
- **后端**: Node.js / Express 
- **通信**: ws (WebSocket) / axios 
- **存储**: better-sqlite3 (WAL 模式)
- **进程管理**: PM2
